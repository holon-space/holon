## Configure `sem` as default Git difftool

Set up the `sem` merge tool to be Git's default difftool, translating configuration from a jj setup.

- Translated jj's `[merge-tools.sem]` block into Git's `[difftool "sem"]` format
- Configured the difftool command: `sem diff --verbose "$LOCAL" "$REMOTE"`
- Set `diff.tool = sem` as the default difftool
- Disabled per-file confirmation prompts with `difftool.prompt = false`

**Status**: `git difftool` now invokes `sem` by default without interactive prompts.
